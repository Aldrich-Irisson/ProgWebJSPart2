  1. Mostrar el valor del séptimo elemento del arreglo f.

    document.writeln(f[6]);

    var g = [8,8,8,8,8];

  3. Calcular el total de los elementos del arreglo c, que contiene 100 elementos numéricos.

    var total = 0;
    for ( var a in c ){
    total += c[a];
    }

  4. Copiar el arreglo a de 11 elementos en la primera porción del arreglo b, que contiene 34 elementos.

    for( var x in a ){
    b[x] = a[x];
    }

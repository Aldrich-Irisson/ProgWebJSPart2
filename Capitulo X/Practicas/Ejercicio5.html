Ejercicios de autoevaluación

10.5 Encuentre el error en cada uno de los siguientes segmentos de código, y explique cómo corregirlo:


  1. var b = new Array(10);
    var longitud = b.length;
    for ( var i = 0; i <= longitud; ++i ){ 
       b[ i ] = 1;
    }

    ERROR: Error en la condición i <= longitud, ya que no existe el índice 10 en el arreglo (sólo es de 0 a 9):
    Corrección: Cambiar el operador <= a <
    var b = new Array(10);
    var longitud = b.length;
    for ( var i = 0; i < longitud; ++i ){ 

       b[ i ] = 1;
    }

  2. var a = [ [1, 2], [3, 4] ];
    a[1, 1] = 5;

    ERROR: El indexado de arreglos se realiza de manera incorrecta.
    Corrección: Cambiar la instrucción a a[1][1] = 5
    var a = [ [1, 2], [3, 4] ];
    a[1][1] = 5;

Ejercicios de autoevaluación

10.1 Complete las siguientes oraciones:


  1. Es posible almacenar listas y tablas de valores en ___arreglos___.
  2. El número que se utiliza para referirse a un elemento específico de un arreglo se conoce como su ___índice___.
  3. El proceso de poner en orden los elementos de un arreglo se conoce como ___ordenamiento___ el arreglo.
  4. Al proceso de determinar si un arreglo contiene cierto valor clave se le conoce como ___búsqueda___ el arreglo.
  5. Un aneglo que utiliza dos índices se conoce como arreglo ___bidimensional___.

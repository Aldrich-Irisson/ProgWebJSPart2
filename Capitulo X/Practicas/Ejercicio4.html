Ejercicios de autoevaluación

10.4 Escriba instrucciones de JavaScript para realizar cada una de las siguientes tareas:


  1. Declarar y crear el arreglo con tres filas y tres columnas.

    var arreglo = [
      [0, 0, 0],
      [0, 0, 0],
      [0, 0, 0]
    ];

  2. Almacenar el número total de elementos en la variable totalElementos.

    var totalElementos = arreglo.length * 3;

  3. Usar una instrucción for...in para inicializar cada elemento del arreglo con la suma de sus índices de fila y de columna. Suponga que las variables x y y declaran como variables de control.

    for ( var x in arreglo ){
      for ( var y in arreglo[x] ){
        arreglo[x][y] = x + y;
      }
    }

Ejercicios de autoevaluación

11.1 Complete las siguientes oraciones:


  1. Como JavaScript usa objetos para realizar muchas tareas, se conoce por lo común como ___lenguaje de programación basado en objetos___.
  2. Todos los objetos tienen ___atributos___ y exhiben ___comportamientos___.
  3. Los métodos del objeto ___Math___ nos permiten realizar muchos cálculos matemáticos comunes.
  4. Al proceso de invocar (o llamar) al método de un objeto se le conoce como ___enviar un mensaje al objeto___.
  5. Las literales de cadena o constantes de cadena se escriben como una secuencia de caracteres en ___comillas dobles___ o ___comillas sencillas.___.
  6. Los índices para los caracteres en una cadena empiezan en ___( )___
  7. Los métodos ___indexOf___ y ___lastIndexOf___ de String buscan la primera y última ocurrencias de una subcadena en un objeto String, respectivamente.
  8. Al proceso de dividir una cadena en tokens se le denomina ___tokenización___.
  9. El procesamiento de fechas y horas puede realizarse con base en la ___zona horaria local de la computadora___ o en el ___Tiempo Universal Coordinado (UTC)___ del Estándar de Tiempo Mundial.
  10. El método ___parse___ de Date recibe como argumento una cadena que representa una fecha y hora, y devuelve el número de milisegundos transcurridos entre la medianoche del 1 de enero de 1970 y la fecha y hora especificadas.
  11. Las aplicaciones Web pueden usar la propiedad ___localStorage___ del objeto window para almacenar hasta varios megabytes de datos de cadenas de pares clave/valor en la computadora del usuario y pueden acceder a estos datos a través de sesiones y pestañas del navegador.
  12. Las aplicaciones Web que necesitan acceso a los datos de pares clave/valor para sólo una sesión de navegación y que deben mantener esos datos separados entre varias pestañas pueden usar la propiedad ___sessionStorage___ del objeto window.
  13. Un selector ___:first-child___ de CSS3 selecciona el primer hijo de un elemento.
  14. Un selector ___nth-child___ de CSS3 con el argumento "odd" selecciones los elementos hijos impares y uno con el argumento "even" selecciona los elementos hijos pares.
